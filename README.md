# 🔧 Rolim Labs — Portfólio

Site de portfólio pessoal da **Rolim Labs**, desenvolvido com HTML, CSS e JavaScript puro. Design moderno, dark theme e totalmente responsivo.

## ✨ Funcionalidades

- **Cursor personalizado** com animação suave
- **Hero animado** com grid, glow e estatísticas flutuantes
- **Marquee** de tecnologias animado
- **Seção Sobre** com timeline de carreira
- **Serviços** em grid com hover interativo
- **Números de impacto** com destaque visual
- **Projetos** em destaque com layout assimétrico
- **Clientes** com cards e resultados alcançados
- **Formulário de contato** funcional via Formspree
- **Scroll reveal** em todos os elementos
- **Totalmente responsivo** (mobile-first)

## 🛠 Tecnologias

- HTML5 semântico
- CSS3 (variáveis, grid, animações, clip-path)
- JavaScript vanilla (sem frameworks)
- [Formspree](https://formspree.io) — envio de formulário
- [Google Fonts](https://fonts.google.com) — Syne + DM Mono

## 🚀 Como usar

1. Clone o repositório:
```bash
git clone https://github.com/seu-usuario/rolimlabs-portfolio.git
```

2. Abra o arquivo no navegador:
```bash
open index.html
```

> Nenhuma dependência ou build necessário. É um arquivo HTML único.

## 📬 Formulário de Contato

O formulário utiliza o [Formspree](https://formspree.io). Para configurar com seu próprio email:

1. Crie uma conta em formspree.io
2. Crie um novo form e copie o endpoint gerado
3. Substitua a URL no HTML:
```html
action="https://formspree.io/f/SEU_ID_AQUI"
```
